New Hope Fish Dictionary

  • Fish Dictionary
  • How It Works

Summary:

This app works by making calls to the Animal Crossing: New Horizons API provided by Alexis. It makes a call to the API to count the current number of fish available. It sets the slider's maximum number to that value. It then takes a fish ID, provided by the user as input. It makes a call to the API using the fish ID to get the information for that fish. Finally, it displays the fish's ID, name, and picture to the user.

How To:

  1. Select a fish ID either by entering the ID or dragging the slider.
  2. Press the "Go!" button
